<HTML 이론 정리>
*기본 구조*
HTML:Hyper Text Markup Language의 약자이다.
     웹페이지는 HTML 문서라고도 부르며, HTML 태그들로 구성된다.
     각각의 HTML 태그는 웹페이지의 디자인이나 기능을 결졍하는데 사용된다.
<!DOCTYPE html>: 현재 문서가 HTML5 문서임을 명시
<html>: HTML문서의 루트 요소를 정의함.
<head>: HTML 문서의 메타데이터를 정의한다.
                    ㄴ> HTML문서에 대한 정보로 웹브라우저에는 직접적으로 표현되지 않는 정보를 의미함.
                        메타데이터는 <title>, <style>, <meta>, <link>, <scipt>, <base>태그 등을 이용하여 표현할수 있음.
<title>: HTML 문서의 제목을 정의하며, 다음과 같은 용도로 사용됨.
         ㄴ>웹 브라우저의 툴바에 표시
            웹 브라우저의 즐겨찾기에 추가할 때 즐겨찾기의 제목이 됨.
            검색 엔진의 결과 페이지에 제목을 표시   
<body>: 웹 브라우저를 통해 보이는 내용 부분
<h1> ~ <h6> :가장 큰 <h1>태그부터 가장 작은 <h6>태그까지 다양한 크기로 제목을 표현할 수 있습니다.
<p>: 단락을 나타냄                               
태그와 요소: 태그는 HTML 페이지에서 객체를 만들 때 사용하는것
            또한 태그를 사용해 만들어진 객체를 요소라고 부른다.
ex) <p class = "para">  test </p>
     ㄴ> p = 태그이름
        class = 속성명
        "para" = 속성값
        test = 내용
        </p> 종료태그
속성: 태그에 추가 정보를 부여할 때는 속성을 사용한다.
